# 社区行为准则

## 我们的承诺

作为这个项目的成员、贡献者和维护者，我们承诺为所有人提供一个友好、安全和欢迎的环境，无论其年龄、体型、可见或不可见的残疾、种族、性别特征、性别认同和表达、经验水平、教育程度、社会经济地位、国籍、个人外貌、种族、宗教或性取向如何。

我们承诺以有助于建立开放、友好、多元、包容和健康社区的方式行动和互动。

## 我们的标准

有助于创建积极环境的行为示例包括：

- 使用友好和包容的语言
- 尊重不同的观点和经验
- 优雅地接受建设性批评
- 关注对社区最有利的事情
- 对其他社区成员表示同理心

不可接受的行为示例包括：

- 使用带有性暗示的语言或图像，以及不受欢迎的性关注或性挑逗
- 发表挑衅、侮辱或贬损性的评论，以及人身或政治攻击
- 公开或私下骚扰
- 未经明确许可发布他人的私人信息，如物理或电子邮件地址
- 其他在专业环境中被认为不适当的行为

## 执行责任

社区维护者有责任明确行为准则的标准，并应对任何他们认为不适当、威胁、冒犯或有害的行为采取适当和公正的纠正措施。

社区维护者有权利和义务删除、编辑或拒绝不符合本行为准则的评论、提交、代码、wiki 编辑、问题和其他贡献，并在适当情况下暂时或永久禁止任何贡献者的其他不当行为、威胁、冒犯或有害的行为。

## 适用范围

本行为准则适用于所有项目空间，也适用于个人在公共空间代表项目或其社区时。代表项目或社区的示例包括使用官方项目电子邮件地址、通过官方社交媒体帐户发布内容，或在线上或线下活动中担任指定代表。

## 执行

如遇滥用、骚扰或其他不可接受的行为，可通过 GitHub Issues 或邮件联系项目维护团队。所有投诉都将被及时和公正地审查和调查。

所有社区维护者有义务尊重任何事件报告者的隐私和安全。

## 执行指南

社区维护者在确定任何被认为违反本行为准则的行为的后果时将遵循这些社区影响指南：

### 1. 纠正

**社区影响**：使用不适当的语言或其他在社区中被认为不专业或不受欢迎的行为。

**后果**：社区维护者发出的私人书面警告，明确违规的性质，并解释为什么该行为不适当。可能要求公开道歉。

### 2. 警告

**社区影响**：通过单一事件或一系列行为的违规。

**后果**：对已继续行为的警告。在指定时间内不得与相关人员互动，包括主动与执行行为准则的人员互动。这包括避免在社区空间以及外部渠道（如社交媒体）中的互动。违反这些条款可能导致临时或永久封禁。

### 3. 临时封禁

**社区影响**：严重违反社区标准，包括持续的不当行为。

**后果**：在指定时间内临时禁止任何形式的互动或公开交流。在此期间，不允许与相关人员进行公开或私人互动，包括主动与执行行为准则的人员互动。违反这些条款可能导致永久封禁。

### 4. 永久封禁

**社区影响**：表现出违反社区标准的模式，包括持续的不当行为、骚扰个人，或对个人类别进行攻击或贬低。

**后果**：永久禁止在项目社区内进行任何形式的公开互动。
